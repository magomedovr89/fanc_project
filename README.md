# FANC RD 
Сервис по сбору данных (Фотографий) для дальнейшей аналитики.

Сервис представляет собой телеграмм бот для сбора информации и сайт на Flask валидации фотографий.

## Что будет реализовано:
- Бот
  - Регистрация пользователя
    - Для ведения рейтинга пользователя
    - Для авторизации пользователя на сайте
  - Передача фотографий
  - Получение справочной информации
- Сайт
  - Авторизация пользователей через телеграм
  - Обработка фотографий (Деление на категории)

Функции которые будут добавлены в дальнейшем:
- &#9989; Сохранение уникального идентификатора чата в базе данных
  - Реализовать возможность сохранения времени сессий
- &#10060; Создание декоратора для ведения логов бота и сайта
- &#9989; Класс для работы с базой данных
  - сделать более лаконичные и простые функции
- &#9989; Реализовать возможность заполнения первичной базы данных
  - Искать новые возможности для улучшения 
- &#10060; Реализовать связь между локальной и центральной базой данных
- &#9989; Добавление времени начала сессии с пользователем
- &#10060; Добавление времени активности пользователя (Отправка фото, запрос информации)
- &#9989; Добавление функции пожертвований для оплаты вычислений и работы экспертов
